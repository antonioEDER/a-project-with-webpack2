<h2 align="center">Install</h2>

Build Development with yarn:

```bash
yarn build
```

Build Prod with yarn:

```bash
yarn build:prod
```

Start with yarn:

```bash
yarn start
```

Serve with webpack-dev-serve:

```bash
yarn serve
```

Serve with express Development:

```bash
yarn express
```

Serve with express Prod:

```bash
yarn express:prod
```
